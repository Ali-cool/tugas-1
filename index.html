1.Web 1.0 (Web Statis)-"baca saja"

Ciri-ciri:

  • Hanya bisa di baca pengguna, tidak bisa interaktif
  • Konten dibuat oleh pemilik web, pengunjung hanya bisa melihat.
  • Tidak ada login
  • HTML digunakan murni untuk tampilan konten.

2.Web 2.0 (Web Sosial)-“Baca dan Tulis”

Ciri-ciri:

  • Pengguna bisa berinteraksi: komentar, upload konten, buat akun.
  • Munculnya media sosial, blog, forum.
  • Ada penggunaan JavaScript, CSS, dan database.
  • HTML lebih dinamis dan digabung dengan backend>

3.Web 3.0 (Web Semantik)-“Baca, Tulis, dan Cerdas”

Ciri-ciri:

  • Menggunakan data terstruktur dan AI.
  • Mesin bisa “mengerti” konten (bukan cuma menampilkan).
  • Contoh: asisten virtual, rekomendasi otomatis.
  • Ada penggunaan semantic tag HTML5, JSON-LD, dll.

4.Web 4.0 (Web Cerdas & Terhubung)-“Web Ubiquitous”

Ciri-ciri:

  • Web terhubung dengan perangkat pintar (IoT).
  • Real-time interaction, AI, dan pemrosesan otomatis.
  • User e>xperience sangat personal dan adaptif.
  • Teknologi: AI, Big Data, Machine Learning, IoT.

tentang perkembangan Web 1.0 sampai Web 4.0:

1.Web 1.0 - Web Statis (1990-an)

  • Hanya bisa dibaca.
  • Konten dibuat oleh pemilik web.
  • Tidak ada interaksi pengguna.
  • Contoh: halaman HTML sederhana.

Web 2.0 - Web Sosial (2000-an)

  • Pengguna bisa berinteraksi dan berkontribusi.
  • Ada komentar, upload, media sosial, blog.
  • Tidak ada interaksi pengguna.
  • Contoh: halaman HTML sederhana.
